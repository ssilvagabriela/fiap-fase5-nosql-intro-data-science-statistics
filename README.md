# fiap-fase5-nosql-intro-data-science-statistics
Projeto da Fase 5 – NoSQL &amp; Intro Data Science Statistics (FIAP). Inclui provas de conceito para escolha de bancos NoSQL (Grafo, Colunar e Documento) e análise estatística completa de vendas com Python, envolvendo outliers, comparação de médias, correlações e visualizações. Baseado no capítulo “Potencializando o Desempenho com NoSQL”.
